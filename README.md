# Planet omkreds

Dette program er skrevet i C og bruges til at øve:

- **Enums** (enumeration) til at repræsentere planeter
- **Switch-case** til at håndtere input og udskrift
- **Funktioner** til at gøre koden modulær
  
## Hvordan det virker

1. Programmet viser en liste over planeter med et bogstav for hver.
2. Brugeren indtaster et bogstav (m/v/e/a/j/s/u/n).
3. Programmet udskriver planetens omkreds i km.
